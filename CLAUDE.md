# Panduan Proyek E-Perpus

## Aturan Penulisan Kode

### Dilarang Menggunakan Emoji
Jangan gunakan emoji di manapun dalam kode

Gunakan **SVG icon** (Heroicons atau sejenisnya) sebagai pengganti emoji di UI.

### Bahasa
Selalu gunakan Bahasa Indonesia dalam semua komunikasi dan respons.

## Konvensi Kode
- Gunakan `cont` class untuk container utama (sudah didefinisikan di `app.css`)
- Warna utama: `#DC2626` (primary red), tersedia via `text-primary`, `bg-primary`
- Font heading: Plus Jakarta Sans (`font-heading`)
- Font body: Inter (`font-sans`)
